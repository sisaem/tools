PDF ↔ 이미지 변환 가이드

PDF와 이미지는 비슷해 보이지만 용도가 다릅니다. PDF는 여러 페이지 문서를 보관하고 공유하는 데 적합하고, 이미지는 특정 페이지나 화면을 빠르게 보여줄 때 편리합니다.

이 안내는 일반적인 PDF 작업을 돕기 위한 설명입니다. 중요한 문서는 원본을 별도로 보관하고, 결과 파일을 직접 확인한 뒤 사용하세요.

PDF를 이미지로 변환하는 경우

PDF의 특정 페이지를 메신저나 문서에 붙여 넣어야 할 때 이미지 변환이 편리합니다. 예를 들어 안내문 한 페이지, 표지 이미지, 캡처용 자료를 만들 때 사용할 수 있습니다. 다만 이미지로 변환하면 텍스트 선택이나 검색은 어려워질 수 있습니다.

이미지를 PDF로 변환하는 경우

여러 장의 이미지 영수증, 스캔 이미지, 촬영 문서를 하나의 PDF로 정리해야 할 때 유용합니다. 이미지 파일 이름을 순서대로 정리한 뒤 변환하면 페이지 순서 오류를 줄일 수 있습니다.

JPG와 PNG 차이

JPG는 사진처럼 색이 많은 이미지에 적합하고 용량이 비교적 작습니다. PNG는 글자, 선, 화면 캡처처럼 경계가 뚜렷한 이미지에 적합하지만 용량이 커질 수 있습니다. 문서 스캔 이미지는 상황에 따라 둘 다 사용할 수 있으므로 결과 용량과 선명도를 비교해 보세요.

해상도와 용량

해상도가 높을수록 글자는 선명하지만 파일 용량이 커집니다. 이메일이나 웹 업로드용이라면 너무 큰 해상도는 불편할 수 있습니다. 반대로 인쇄용이나 보관용이라면 지나치게 낮은 해상도를 피하는 것이 좋습니다.
