개인정보처리방침

Sisaemdal Tools는 회원가입 없이 사용할 수 있는 브라우저 기반 웹 도구입니다. 사이트 이용 과정에서 운영자가 직접 사용자의 PDF, 이미지, Excel 파일 원본을 서버에 저장하거나 열람하는 기능은 제공하지 않습니다.

수집하는 정보

사이트 자체는 이름, 이메일, 전화번호 같은 개인정보 입력을 요구하지 않습니다. 문의를 GitHub Issues로 남기는 경우 사용자가 직접 작성한 내용과 GitHub 계정 정보가 GitHub 정책에 따라 공개될 수 있습니다.

파일 처리

PDF 도구의 주요 작업은 사용자의 브라우저에서 처리되도록 구성되어 있습니다. 사용자가 선택한 파일은 작업을 위해 브라우저 메모리에서 읽힐 수 있으며, 작업이 끝난 뒤 결과 파일을 사용자가 직접 다운로드합니다. 민감한 문서는 공용 PC나 신뢰할 수 없는 브라우저 환경에서 사용하지 않는 것이 좋습니다.

외부 서비스

이 사이트는 광고 게재와 사이트 분석, 라이브러리 로딩을 위해 Google AdSense 및 외부 CDN 스크립트를 사용할 수 있습니다. 이 과정에서 브라우저, 기기, 쿠키, 광고 식별자 등 일반적인 웹 이용 정보가 각 서비스의 정책에 따라 처리될 수 있습니다.

문의

개인정보 관련 문의나 오류 제보는 문의 페이지의 GitHub Issues 링크를 통해 남길 수 있습니다. 공개 게시판이므로 민감한 정보는 포함하지 마세요.
