문의 및 오류 제보

Sisaemdal Tools에 대한 오류 제보, 기능 제안, 문서 내용 수정 의견은 GitHub Issues를 통해 남길 수 있습니다.

주의: GitHub Issues에 작성한 내용은 공개될 수 있습니다. 개인정보, 비밀번호, 원본 PDF 파일, 내부 문서, 민감한 이미지는 업로드하지 마세요.

문의 전에 확인하면 좋은 항목

  • 문제가 발생한 페이지 주소
  • 사용한 브라우저 종류와 버전
  • 파일 크기와 PDF 암호 여부
  • 재현 가능한 간단한 설명

파일 원본을 공유하지 않아도 문제 상황을 설명할 수 있는 범위에서만 작성해 주세요.
