PDF 병합·분할 가이드

PDF 병합과 분할은 가장 자주 쓰이는 문서 정리 작업입니다. 회의 자료, 스캔 문서, 제출 서류처럼 여러 파일로 흩어진 문서를 하나로 모으거나 필요한 페이지만 분리할 때 유용합니다.

이 안내는 일반적인 PDF 작업을 돕기 위한 설명입니다. 중요한 문서는 원본을 별도로 보관하고, 결과 파일을 직접 확인한 뒤 사용하세요.

병합이 필요한 상황

여러 부서에서 받은 PDF 자료를 하나의 회의 자료로 만들거나, 영수증과 신청서처럼 함께 제출해야 하는 문서를 한 파일로 묶을 때 병합 기능을 사용할 수 있습니다. 병합 전에는 파일 이름을 01, 02, 03처럼 순서대로 정리해 두면 결과 순서를 확인하기 쉽습니다.

분할이 필요한 상황

전체 문서 중 일부 페이지만 공유해야 하거나, 스캔된 여러 문서가 하나의 PDF로 합쳐진 경우 분할 기능이 필요합니다. 민감한 정보가 포함된 페이지를 제외하고 공유해야 할 때도 분할 후 결과 파일을 반드시 다시 열어 확인하는 것이 중요합니다.

페이지 순서 확인

PDF 병합에서 가장 많이 발생하는 실수는 페이지 순서 오류입니다. 결과 파일을 만든 뒤 첫 페이지, 중간 페이지, 마지막 페이지를 빠르게 확인하세요. 제출용 문서라면 목차나 표지, 첨부자료 순서가 맞는지도 확인해야 합니다.

파일 용량 관리

여러 스캔 PDF를 합치면 파일 용량이 커질 수 있습니다. 이메일 첨부 제한이나 업로드 제한이 있는 경우 병합 전에 이미지 해상도와 페이지 수를 고려하세요. 용량이 큰 파일은 전송 전에 열리는 속도와 업로드 가능 여부를 확인하는 것이 좋습니다.
