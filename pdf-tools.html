무료 온라인 PDF 편집·변환 도구

PDF Tools는 별도 프로그램 설치 없이 브라우저에서 PDF 파일을 정리하고 변환할 수 있도록 만든 무료 웹 도구입니다. PDF 암호 설정과 해제, 여러 PDF 병합, 페이지 분할, 회전, PDF와 이미지 변환, 기본 표 중심의 PDF·Excel 변환 기능을 제공합니다.

개인정보 보호 안내: 선택한 파일은 사용자의 브라우저 안에서 처리되며, 이 정적 사이트는 파일을 서버로 업로드하도록 설계되어 있지 않습니다. 민감한 문서는 처리 후 결과 파일을 안전한 위치에 보관하세요.

🔐

PDF 암호 관리

PDF 파일의 암호를 설정하거나 해제하여 보안을 관리하세요
📑

PDF 병합/분할

여러 PDF를 하나로 병합하거나, 하나의 PDF를 여러 개로 분할하세요
🔄

PDF 회전

PDF 페이지를 원하는 각도로 회전하고 미리보기로 확인하세요
🔀

PDF ↔ 이미지

PDF를 이미지로 변환하거나, 이미지를 PDF로 변환하세요
📊

PDF ↔ Excel

PDF를 Excel로 변환하거나, Excel을 PDF로 변환하세요 (기본 표 구조만 지원)

제공 기능과 사용 안내

  • PDF 암호 관리: PDF에 암호를 설정하거나, 사용자가 알고 있는 암호를 입력해 암호 보호를 해제할 수 있습니다.
  • PDF 병합/분할: 여러 파일을 하나로 합치거나 페이지 단위, 범위 단위, N페이지 단위로 나눌 수 있습니다.
  • PDF 회전: 문서 방향이 잘못된 PDF를 미리보기로 확인하며 90도 또는 180도로 회전할 수 있습니다.
  • PDF ↔ 이미지: PDF 페이지를 JPG/PNG 이미지로 저장하거나 여러 이미지를 하나의 PDF로 만들 수 있습니다.
  • PDF ↔ Excel: 기본 표 구조를 가진 PDF를 Excel로 변환하거나 단순 Excel 데이터를 PDF 표로 변환할 수 있습니다.

자주 묻는 질문

파일이 서버에 업로드되나요?

아니요. 현재 제공되는 도구는 브라우저에서 동작하는 정적 웹 앱 형태이며, 사용자가 선택한 파일을 별도 서버에 저장하거나 전송하는 기능을 포함하지 않습니다.

암호화된 PDF도 모든 기능에서 사용할 수 있나요?

암호가 필요한 PDF는 먼저 암호 관리 기능에서 올바른 암호를 입력해 해제한 뒤 병합, 분할, 회전, 변환 기능을 사용하는 것을 권장합니다.

복잡한 PDF 표도 Excel로 정확히 변환되나요?

아니요. PDF에서 Excel로 변환하는 기능은 기본적인 표 구조를 대상으로 하며, 병합 셀·복잡한 레이아웃·이미지 기반 표는 정확도가 낮을 수 있습니다.
