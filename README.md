# Sisaemdal Tools

Sisaemdal Tools는 브라우저에서 바로 사용할 수 있는 무료 온라인 웹 도구 모음입니다. PDF, 이미지, 텍스트, QR 코드 도구와 사용 가이드를 제공합니다.

## 제공 기능

- PDF 암호 설정 및 해제
- PDF 병합, 분할, 회전
- PDF와 이미지/Excel 간 기본 변환
- 이미지 크기 조정, 압축, 포맷 변환
- 텍스트 글자 수 계산, 줄 정리, 중복 제거
- QR 코드 생성 및 PNG 저장

문의와 오류 제보는 GitHub Issues를 이용해 주세요. 개인정보, 비밀번호, 원본 파일은 포함하지 마세요.
